Essential Gutenberg blocks for WordPress 5.

== Description ==

AceBlocks is a Gutenberg compatible collection of simple and beautiful blocks for WordPress 5. Use AceBlocks to create custom icon cards, image blocks, testimonials, social icons, and more!

> AceBlocks is under active development. We're keeping up with the quick pace of Gutenberg's newest features, so things can (read: will) change at any time!

## AceBlocks currently includes the following blocks:

| Block Type            | Description                           |
| --------------------- | ------------------------------------- |
| Testimonial           | Inline quotes from your customers     |
| Alert                 | Display important notices and errors  |
| Icon Block            | Show a message with an icon           |
| Social Block          | Link out to your top social profiles  |
| Image Block           | Display content with an image         |

== Installation ==

This plugin can be installed directly from your site. It requires the Gutenberg plugin until that gets merged into WordPress core in WordPress 5.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "AceBlocks" into the Search and hit Enter.
3. Locate the AceBlocks plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.

It can also be installed manually.

1. Download the AceBlocks plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate AceBlocks in the list and click the *Activate* link.

== Frequently Asked Questions ==

= Can AceBlocks be used with any theme? =

Yes, you can use AceBlocks with any theme. Really!

= Do I need Gutenberg to use AceBlocks? =

Yes, you will need to install the [Gutenberg plugin](https://wordpress.org/plugins/gutenberg/) to use AceBlocks. Eventually, Gutenberg (the block editor) will be merged into WordPress and you won't need the plugin.

= When is Gutenberg going to be merged into core? =

Gutenberg is scheduled to be merged into WordPress when version 5.0 becomes available. The release date is still tentative, so check back soon!
